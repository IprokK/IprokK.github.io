Проект создавался недавно. Написан на ЯП Python. Использован API Телеграм бота. Умеет взаимодействовать с Mysql, куда и ведёт сохранение данных. Бот умеет сохранять опросы и потом публиковать их в нужные чаты. Кроме того, он ведёт подсчёт баллов и составляет рейтинг участников. Для исключения накрутки баллов, доступ к управлению ботом ограничен (он принимает сообщения только от определнных аккаунтов). Бот используется по сей день.
 
При выгрузке кода на Git все личные данные заменил нулями.
